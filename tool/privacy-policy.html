API Integrator - Data Protection and Privacy

Privacy Policy for API Integrator

Important Notice: API Integrator is a client-side application that runs entirely in your web browser. All data processing and storage occur locally on your device.

1. Data Collection and Processing

1.1 What Data We Collect

API Integrator is designed to operate with minimal data collection. The application:

  • Does NOT collect personal information such as name, email, or IP address
  • Does NOT transmit data to external servers except for the APIs you explicitly configure
  • Stores configuration data locally in your browser's localStorage
  • Only processes data that you provide for API integration purposes

1.2 Local Data Storage

The application stores the following information locally in your browser:

  • API source configurations (endpoints, headers, authentication details)
  • Field mappings between different APIs
  • Generated code snippets and integration settings

This data remains on your device and is never transmitted to our servers.

2. API Communications

2.1 Direct API Interactions

When you use API Integrator to test connections or generate integration code:

  • Requests are made directly from your browser to the APIs you configure
  • We do not act as a proxy or intermediary for these communications
  • API credentials and data flow directly between your browser and your chosen APIs
  • We cannot access, view, or store your API responses or authentication tokens

2.2 Your Responsibility

You are responsible for:

  • Ensuring you have proper authorization to access the APIs you configure
  • Protecting your API keys, tokens, and credentials
  • Complying with the terms of service of the APIs you use
  • Handling data in accordance with applicable privacy laws and regulations

3. Browser Storage and Security

3.1 LocalStorage Usage

Configuration data is stored using browser localStorage, which means:

  • Data persists between browser sessions on the same device
  • Data is accessible only to the API Integrator application on your browser
  • Data is not encrypted by default (browser-dependent)
  • Data can be cleared by clearing browser data or using the application's clear functions

3.2 Security Considerations

Important security notes:

  • API keys and tokens stored in localStorage are accessible to anyone with physical access to your browser
  • Shared computers may expose your stored configurations
  • Browser extensions could potentially access stored data
  • We recommend using environment-specific API keys when possible

4. Third-Party Services

4.1 External Links

The application may contain links to:

  • GitHub repository for source code
  • Documentation and support resources
  • Developer resources and API documentation

These external sites have their own privacy policies which we encourage you to review.

4.2 No Analytics or Tracking

API Integrator does not use:

  • Google Analytics or similar tracking services
  • Cookies for tracking or analytics purposes
  • Third-party tracking pixels or beacons
  • User behavior monitoring or heatmaps

5. Data Rights and Control

5.1 Your Data Rights

You have complete control over your data:

  • Access: All stored data is visible in the application interface
  • Modification: You can edit or update any stored configuration
  • Deletion: You can delete individual sources or clear all data
  • Export: You can copy configurations for backup or migration

5.2 Data Deletion

To remove all stored data:

  • Use the "Clear All Sources" function in the application
  • Clear your browser's localStorage/site data
  • Uninstall or remove the application from your browser

6. Children's Privacy

API Integrator is not intended for use by children under 13 years of age. The application does not knowingly collect personal information from children. If you believe a child has provided information to us, please contact us immediately.

7. International Data Transfers

Since API Integrator operates entirely within your browser and does not collect or transmit data to our servers, there are no international data transfers involving our systems. However, when you configure APIs located in different countries, your data will be transferred according to those services' policies and applicable laws.

8. Changes to This Privacy Policy

We may update this privacy policy from time to time. Changes will be posted on this page with an updated revision date. Significant changes will be highlighted in the application interface.

9. Contact Information

If you have questions about this privacy policy or concerns about data handling, please contact:

  • Website: anastasios-bolkas.tech
